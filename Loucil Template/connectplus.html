Connect+



Connect+ was my very first big project that I worked on during my first semester as an IT major. It is a social networking site, that's like a play on Google+, which I built on a cloud-based IDE called Codio for my class CGS2835 Intro to Web Development. The site enables the user to create a profile, upload a profile picture, write posts and request to be friends with other users.

It was built without a framework and utilizes HTML, CSS, PHP, MySQL and Javascript, utilizing the sample code that our professor Geo Miller provided. I modified and added onto the code to make it look like how it looks now. I tested the application by creating a profile for every character in the show How I Met Your Mother, as well as posting a status of their most famous quotes from the show. This was my first experience developing a dynamic site, so I decided to go all out and really channel my creative side. You can take a look at the source code here.
